loop types
1. while loop
    -->top tested
    --> add count variable outside
    --> check condition
    --> increment/decrement of count variable
2. for loop
      -->it is easiest loop
      -->it is top tested loop
      --> count variable declaration, condition check , increment declaration of count variable
          is done at single place

          syntax
           for(counter; condition;increment,decrement)

3. do while loop
    --> bottom tested loop
    --> it will execute at least once
        if true condition it may execute one or more times
        if false condition it will execute one times
