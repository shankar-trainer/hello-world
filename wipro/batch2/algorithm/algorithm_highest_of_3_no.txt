ALGORITHM TO FIND HIGHEST OF THREE NUMBER
STEP1. START 
STEP2. INPUT A,B,C
STEP3. IF A>B 
            THEN IF A>C 
			    THEN G←A
			ELSE 
			    G←C
			END IF 
       ELSE
	    IF B>C
		  THEN G←B
		ELSE 
               G←C 		
       END IF
STEP4  PRINT"HIGHEST IS ":G
STEP5. END
