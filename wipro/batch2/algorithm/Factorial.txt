Program Factorial 
one way 1*2*3*4*5

Start
 Declare  counter, fact,n as Number 

 counter=1
 fact=1
 n=0

 Print "Enter Number"
 read n

 Repeat 
  fact =fact*counter
  counter=counter+1
 Until counter<=n 

 Print "Factorial is",fact 
End
