
in java we tow data type

primitive   no attribute , no method
each primitive  associated with reference data type(wrapper class )
===============================================
primitive          reference data type
int                 java.lang.Integer
long                java.lang.Long
char                Character
float               java.lang.Float

=========================

constructor are not inherited
each constructor is calling super class constructor by the
super keyword

super()--- is added in each  of constructor by the compiler
